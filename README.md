Organize your files into groups for quick access and better productivity.

## Features

- **Create Groups** - Organize your files into custom groups
- **Quick Access** - Open files directly from the sidebar
- **Easy Management** - Add, remove, rename groups and files
- **Persistent Storage** - All groups are automatically saved to your workspace
- **Multi-language Support** - Available in English and Chinese

## How to Use

### Create a Group

1. Click the **➕** button in the "File Groups" panel
2. Enter a group name
3. The new group appears in the sidebar

### Add Files to a Group

**Method 1: Right-click Menu**
- Right-click on a file in the Explorer
- Select "Add File to Group"
- Choose the target group

**Method 2: Editor Tab**
- Right-click on the editor tab
- Select "Add File to Group"
- Choose the target group

### Open Files

Simply click on any file in the "File Groups" panel to open it in the editor.

### Manage Groups

- **Rename Group**: Click the ✏️ icon next to the group name
- **Delete Group**: Click the 🗑️ icon next to the group name
- **Remove File**: Click the ➖ icon next to the file name

### Refresh

Click the 🔄 button to refresh the file groups view.

## Keyboard Shortcuts

| Command | Shortcut |
|---------|----------|

# 文件分组管理器

将文件组织到分组中以便快速访问，提高工作效率。

## 功能特性

- **创建分组** - 将文件组织到自定义分组中
- **快速访问** - 从侧边栏直接打开文件
- **简单管理** - 添加、删除、重命名分组和文件
- **持久化存储** - 所有分组自动保存到工作区
- **多语言支持** - 支持中文和英文

## 使用方法

### 创建分组

1. 点击"文件分组"面板中的 **➕** 按钮
2. 输入分组名称
3. 新分组出现在侧边栏中

### 添加文件到分组

**方法一：右键菜单**
- 在资源管理器中右键点击文件
- 选择"添加文件到分组"
- 选择目标分组

**方法二：编辑器标签**
- 右键点击编辑器标签
- 选择"添加文件到分组"
- 选择目标分组

### 打开文件

在"文件分组"面板中点击任意文件即可在编辑器中打开。

### 管理分组

- **重命名分组**：点击分组名称旁的 ✏️ 图标
- **删除分组**：点击分组名称旁的 🗑️ 图标
- **移除文件**：点击文件名旁的 ➖ 图标

### 刷新

点击 🔄 按钮刷新文件分组视图。

## 快捷键

| 命令 | 快捷键 |
|------|--------|
| 创建分组 | `Ctrl+Shift+P` → "创建分组" |
| 添加文件到分组 | `Ctrl+Shift+P` → "添加文件到分组" |
| 刷新 | `Ctrl+Shift+P` → "刷新" |
